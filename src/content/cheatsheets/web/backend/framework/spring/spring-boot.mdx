* **Autoconfiguraciû°n**: Spring Boot detecta automûÀticamente las dependencias en tu classpath (ej. si tienes `spring-web`, configura un servidor web embebido).

* **Aplicaciones Standalone**: Permite ejecutar tu aplicaciû°n como un JAR independiente con un comando `java -jar`.

### 3.4. Desarrollo Web (APIs REST)

* **`@RequestMapping("/api/users")`**: Mapea una clase o mûˋtodo a una URL. Puede usarse con mûˋtodos HTTP especûÙficos.

* **`@ResponseStatus(HttpStatus.CREATED)`**: Define el cû°digo de estado HTTP de la respuesta.

  ```java

      @GetMapping

          return productService.findAll();
      }

      @GetMapping("/{id}")

                  .map(ResponseEntity::ok)
                  .orElse(ResponseEntity.notFound().build());
      }

      public Product createProduct(@RequestBody Product product) {
          return productService.save(product);
      }
  }
  ```

### 3.5. Acceso a Datos (JPA y Spring Data JPA)

* **`@Entity`**: Marca una clase como una entidad JPA (mapeada a una tabla de base de datos).

## 8. ÞÏˆ Testing

Spring Boot facilita el testing con su mû°dulo de test.

* **`@SpringBootTest`**: Carga el contexto completo de la aplicaciû°n. Ideal para tests de integraciû°n.

  ```java
  import static org.springframework.test.web.servlet.result.MockMvcResultMatchers.status;

  class ProductControllerTest {
      @Autowired
      private MockMvc mockMvc;

      @Test
      void shouldReturnDefaultMessage() throws Exception {
          mockMvc.perform(get("/api/products"))
                 .andExpect(status().isOk());
      }
  }
  ```
* **`@DataJpaTest`**: Para probar componentes de JPA (repositorios) con una base de datos en memoria.
* **`@MockBean`**: Permite aûÝadir mocks al contexto de la aplicaciû°n para aislar los componentes.

## 10. ÞÀ Buenas PrûÀcticas y Consejos

* **Documentaciû°n de API**: Integra Swagger/OpenAPI (con Springdoc OpenAPI) para documentar tus APIs REST.

* **Monitorizaciû°n**: Usa Spring Boot Actuator y combûÙnalo con herramientas de monitorizaciû°n externas (ej. Prometheus, Grafana).
